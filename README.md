# CMAU Demo

一个基于 Vue 3 + Vite 的单页演示项目，用于展示科研测评、结果解读、服务体系、内容页与用户工作台等核心流程。

## 技术栈

- Vue 3
- Vue Router
- Vite
- Tailwind CSS

## 本地运行

```bash
npm install
npm run dev
```

## 构建预览

```bash
npm run build
npm run preview
```

## 主要页面

- 首页 `/home`
- 测评页 `/assessment`
- 结果页 `/results`
- 工作台 `/dashboard`
- 服务页 `/services`
- 内容页 `/content`
- 案例页 `/cases`
- 咨询页 `/consult`

## 目录说明

- `src/views/`：页面视图
- `src/components/`：复用组件
- `src/composables/`：共享逻辑
- `src/data/`：静态数据
- `src/router/`：路由配置
